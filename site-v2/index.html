The route

从 Agent 到阅读，只留一条短路径。

01

生成

Agent 输出报告、方案、复盘，Markdown 是最自然的交付形态。

02

落地

文件留在本地，不需要账号、同步或复杂工作区。

03

阅读

双击 `.md`，进入清爽阅读视图，再按需要编辑或导出。

Reader principles

默认只读，是一种产品态度。

Agent 生成的文档通常先用于理解、判断和转发。LightMD 把“阅读”放在第一位，减少误改文件的风险。

.md

双击即读

把 Markdown 当成普通文档，而不是每次都打开重型编辑器。

LOCAL

本地优先

不做云同步，不做账号，不把文件送到另一个系统。

EXPORT

轻量导出

阅读完成后，可以导出 PNG、PDF 或 HTML，方便分享。
