Why LightMD

Agent 产出很多 `.md`，阅读不该变重。

LightMD 不做知识库，不做账号，不把本地文件推向云端。它只把 Agent 给你的 Markdown，快速变成舒服的阅读页面。

01

Agent 生成

报告、方案、纪要、复盘，最后常常都是 Markdown。

02

双击打开

把 `.md` 当成普通文档处理，不再先打开重型编辑器。

03

本地阅读

默认只读，专注阅读，需要时再编辑和导出。

What it does

小工具，只做高频动作。

.md

双击即读

支持 `.md`、`.markdown`、`.txt`，打开后直接进入阅读视图。

⌘

Mac 原生

Swift 原生 App，贴近系统窗口、菜单和文件打开方式。

只读

默认只读

先保护原文件。需要修改时，再明确进入编辑模式。

轻量导出

把当前文档导出为 PNG、PDF 或 HTML，方便分享和归档。
